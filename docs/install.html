How to Install R Studio

Installation Steps

It is very important to install in order: first R, then R Studio

  1. Install R from the download link. You can pick which mirror to choose. For example, Duke Mirror: https://archive.linux.duke.edu/cran/

  2. Install R Studio (free desktop version) AFTER you installed R, from the link. You will only use R Studio but need to have both on your computer.

How to Use R Studio

Step 1: Open R Studio from applications folder

Step 2: Create a new project

Step 3: Click on “File/New File/R script”

Step 4: Running commands

Commands in R Studio can be written directly into the console area (bottom left), or in the script. To run a completed command press Control + Enter (Windows/Linux) or Command + Enter (Mac).

Step 5: Install Required Packages

Type the following code in the script and run it:

install.packages("tidyverse")    # data wrangling
install.packages("tidyr")        # data wrangling
install.packages("tidytext")     # text analysis
install.packages("dplyr")        # data wrangling
install.packages("tm")           # topic modeling
install.packages("topicmodels")  # topic modeling
install.packages("quanteda")     # text analysis
install.packages("lubridate")    # dates
install.packages("ggplot2")      # visualizations
install.packages("ggthemes")     # visualizations
install.packages("scales")       # visualizations
install.packages("wesanderson")  # visualizations

Step 6: Load Packages into Library

After installation, load the packages before using them:

library(tidyverse)
library(tidytext)
library(tidyr)
library(dplyr)
library(tm)
library(topicmodels)
library(quanteda)
library(lubridate)
library(ggplot2)
library(ggthemes)
library(scales)
library(wesanderson)

Step 7: Getting Help

For help with any command, type ? followed by the command name and run:

?merge
?ggplot
?install.packages

For keyboard shortcuts in R Studio, see the official documentation.

You’re Ready!

Now you are ready to work in R Studio! If you have any issues with installation, please email the instructor or TA, or come to office hours.

Troubleshooting

Problem: Package installation fails

  • Make sure you’re connected to the internet

  • Try selecting a different CRAN mirror

  • Check if you have write permissions

Problem: R Studio won’t open

  • Make sure R is installed first

  • Try reinstalling R Studio

  • Check system requirements

Problem: Can’t load a package

  • Make sure the package is installed first using install.packages()

  • Check the package name spelling

  • Try reinstalling the package
